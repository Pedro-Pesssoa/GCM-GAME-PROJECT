# CardDev

## Resumo
Componente que exibe um cartão de apresentação de desenvolvedor, incluindo foto, nome, especialidade e links para redes sociais (LinkedIn e GitHub). Utilizado na seção de desenvolvedores do projeto.

## Props

| Prop | Tipo | Descrição |
|------|------|-----------|
| nome | string | Nome completo do desenvolvedor |
| especialidade | string | Área de atuação ou especialidade do desenvolvedor |
| fotoUrl | string | URL da foto de perfil do desenvolvedor |
| linkedinUrl | string | URL do perfil do LinkedIn do desenvolvedor |
| githubUrl | string | URL do perfil do GitHub do desenvolvedor |
