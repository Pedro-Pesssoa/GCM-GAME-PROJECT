# Header

## Resumo
Componente de cabeçalho responsivo do site, contendo logo da aplicação e menu de navegação com links para as diferentes seções da página. Implementa menu hambúrguer para dispositivos móveis.

## Props
Este componente não recebe props externas.

## Estado Interno

| Estado | Tipo | Descrição |
|--------|------|-----------|
| isMenuOpen | boolean | Controla se o menu mobile está aberto ou fechado |

## Funcionalidades
- Logo clicável "QuizPlanet" que redireciona para a home
- Menu de navegação com links para seções: Jogo, Sobre o Jogo, Tecnologias, Desenvolvedores
- Menu hambúrguer responsivo para dispositivos móveis
- Atributos de acessibilidade (aria-label, aria-expanded)
